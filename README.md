# GA4J

GA4J 是使用 Java 开发的遗传算法框架。用户使用框架时，只需要编写与问题相关的少量代码，就可以使用遗传算法解决问题。

使用方法可参考[示例工程](https://github.com/Hifumi123/GA4JExamples)。

性能测试可参考 [Benchmark 工程](https://github.com/Hifumi123/GA4JBenchmark)。

## 已实现的功能

### 遗传算法

* 基本遗传算法
* 标准型并行遗传算法

### 染色体编码

* 二进制编码染色体
* 格雷码编码染色体

### 选择算子

* 比例选择
* 最优保存比例选择

### 交叉算子

* 单点交叉
* 双点交叉

### 变异算子

* 基本位变异

## 计划实现的功能

- [x] 双点交叉
- [x] 多线程支持

## 开发环境

* Java 8
* Eclipse 2020-06 (4.16.0)
